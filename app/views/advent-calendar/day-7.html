Alt text helps people who cannot see images understand their purpose. It must describe the meaning of the image, not just the image itself.

What you’re checking today

You’re checking that meaningful images have appropriate alt text, and decorative images are ignored by screen readers.

Quick 5-minute check

  • right-click an image and select Inspect
  • look at the alt attribute in the HTML
  • ensure the alt text describes the purpose of the image, not unnecessary detail
  • use alt="" for decorative images so screen readers skip them
  • look for common errors such as missing alt, filenames used as alt, or overly long descriptions

The WCAG 2.2 skill behind this

1.1.1 Non-text Content - images must have appropriate text alternatives.

Why this matters

Without alt text, screen reader users lose essential context. Good alt text supports accessibility, search engines and resilience if images fail to load.

Try this bonus check

Temporarily disable images using a browser extension. Can you still understand the page?
