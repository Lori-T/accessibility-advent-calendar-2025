Day 11: Check table structure in the Present Inventory

The Present Inventory Spreadsheet tracks all gifts being prepared for delivery. But one of the tables has been built using bold text and colours instead of proper table headers.

Today you’ll check the table structure to ensure headers are marked up correctly.

Your mission today

Check that tables use semantic headers.

Quick 5-minute check

  • use <th> for column and row headers
  • use scope="col" or scope="row" where needed
  • avoid using <td> with bold text pretending to be headings
  • screen readers rely on proper associations

Test the Present Inventory

One column header below has incorrect markup.

Gift Quantity ❌ Destination
Train set 20 Berlin

What this checks for

1.3.1 info and relationships - tables must expose proper header associations.

Why this matters

Screen reader users rely on table headers to understand which data belongs to which column or row.

Bonus challenge

Use a screen reader to navigate the cells. Which header is not announced?
