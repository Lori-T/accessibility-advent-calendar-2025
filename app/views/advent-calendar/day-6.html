Day 6: Check focus visibility in the Reindeer Health Monitor

The Reindeer Health Monitor helps Santa HQ keep an eye on each reindeer’s wellbeing. But some controls in the interface have no visible focus styles, making keyboard navigation difficult.

Today you’ll review part of the interface to check whether focus is clear and easy to see as you tab around the page.

Your mission today

Check whether interactive elements show a visible focus outline.

Quick 5-minute check

  • every focusable element must show a visible outline
  • the outline must not rely only on colour changes
  • removing focus styles entirely is a WCAG failure
  • GOV.UK focus styles are a good baseline

What this checks for

2.4.7 focus visible - users must be able to see which element is focused.
