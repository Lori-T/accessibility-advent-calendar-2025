Icons and SVGs need clear labelling if they convey meaning. Decorative icons must be hidden from assistive technologies.

What you're checking today

You're checking that meaningful icons have accessible names, and decorative icons are ignored by screen readers.

Quick 5-minute check

  • inspect an icon or SVG using Inspect
  • if the icon conveys meaning, ensure it has a label using visible text or aria-label
  • decorative icons should be hidden using aria-hidden="true"
  • check that icons meet the minimum 3:1 contrast ratio requirement

The WCAG 2.2 skill behind this

1.1.1 Non-text Content and 1.4.11 Non-text Contrast - icons must be accessible and perceivable.

Why this matters

Poorly labelled icons confuse users and prevent some people from completing tasks. Good icon accessibility improves clarity for everyone.

Try this bonus check

Turn on a screen reader and navigate your page. Does it read out only meaningful icons?
