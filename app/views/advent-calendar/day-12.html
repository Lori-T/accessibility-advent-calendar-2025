WAVE is a quick and simple testing tool that highlights accessibility issues directly on your page. It helps you spot errors, contrast problems and structural issues fast.

What you're checking today

You're using WAVE to scan your page for common accessibility issues that can be fixed quickly.

Quick 5-minute check (Web version)

  • go to https://wave.webaim.org
  • enter the URL of the page you want to check
  • review the icons that appear on your page
  • red icons highlight errors
  • yellow icons highlight contrast issues
  • blue and green icons show structural information

Quick 5-minute check (Browser extension)

  • install the WAVE extension for Chrome or Firefox
  • open your page and click the WAVE icon in your toolbar
  • use the sidebar to explore errors and alerts
  • click an icon to see the explanation and the related code

The WCAG 2.2 skill behind this

WAVE checks many WCAG criteria, including contrast, structure, labels and alt text.

Why this matters

WAVE helps catch common issues early, saving time and reducing accessibility debt. It highlights problems visually, making them easier to understand.

Try this bonus check

Run WAVE on both desktop and mobile layouts. Some issues only appear at smaller screen sizes.
