Day 5: Check keyboard access on the Gift Tracker

The Department of Elfish Affairs has received a report that the Gift Tracker - the system elves use to update toy progress - can only be used properly with a mouse. This makes it inaccessible for many users, including people who rely on keyboard navigation.

Today you’ll check a small section of the Gift Tracker to see which controls can (and cannot) be reached using only a keyboard.

Your mission today

Identify any controls that cannot be accessed using the keyboard.

Quick 5-minute check

  • use the Tab key to move forward through controls
  • use Shift + Tab to move backwards
  • buttons, links and form fields must all receive focus
  • custom widgets must be keyboard operable

Test the Gift Tracker

One control below cannot be accessed with a keyboard.

❌ Custom toggle (mouse only)

What this checks for

2.1.1 keyboard - all functionality must be operable through a keyboard.

Why this matters

Many users rely on keyboard navigation. If controls cannot be reached or operated, they cannot use the service at all.

Bonus challenge

Try the page using only the keyboard. Which control stops you?
