This accessibility statement applies to the Accessibility Advent Calendar 2025.

Using this website

You should be able to:

  • navigate most of the website using only a keyboard
  • zoom in up to 400 percent without content disappearing
  • navigate using assistive technologies such as screen readers
  • use the site on different devices, including mobile phones and tablets

How accessible this website is

We aim to meet the WCAG 2.2 AA standard wherever possible, unless the content is intentionally innacessible for training purposes. This website also links to other websites, resources, and YouTube videos which may not be fully accessible.
