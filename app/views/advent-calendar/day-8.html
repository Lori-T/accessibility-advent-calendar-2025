Turning off CSS helps you see whether your HTML is well structured. If the page is still readable, you’re likely using semantic markup correctly.

What you’re checking today

You’re checking how your page reads without any styling, and whether content remains in a logical order.

Quick 5-minute check

  • install a CSS toggle extension such as “Web Developer” (Firefox/Chrome)
  • use the “Disable styles” option to remove all CSS
  • check if content still appears in a sensible order
  • look for headings, lists, navigation and text that appear out of place
  • ensure important information is still present and readable

The WCAG 2.2 skill behind this

1.3.1 Info and Relationships - structure must be conveyed through semantics, not visual styling.

Why this matters

Good structure supports accessibility, SEO and compatibility with assistive technologies.

Try this bonus check

With CSS disabled, try pressing Tab. Does the focus order still make sense?
