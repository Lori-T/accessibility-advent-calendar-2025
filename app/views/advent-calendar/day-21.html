Tooltips, pop-ups and content that appears when users hover or focus must be stable, dismissible and easy to interact with.

What you're checking today

You're checking that hover or focus content is usable for mouse, keyboard and touch users.

Quick 5-minute check

  • hover over or tab onto a tooltip or pop-up
  • ensure the content stays visible when you move the cursor or focus inside it
  • check there is an easy way to dismiss the content
  • confirm it does not obscure essential information underneath

The WCAG 2.2 skill behind this

1.4.13 Content on Hover or Focus - hover and focus content must be clear and usable.

Why this matters

Content that appears and disappears too quickly can be impossible for some users to interact with, especially those using keyboards or magnifiers.

Try this bonus check

Try accessing the same tooltip using only the keyboard. Is it usable and stable?
