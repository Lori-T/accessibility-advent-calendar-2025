Some users apply custom text spacing to improve readability. Your content should remain functional and readable when text spacing changes.

What you're checking today

You're checking that increased spacing does not cause text to overlap, become cut off or break the layout.

Quick 5-minute check

  • use a text spacing bookmarklet or extension to apply increased spacing
  • test the WCAG recommended spacing
  • line height: 1.5
  • paragraph spacing: 2 times the font size
  • letter spacing: 0.12em
  • word spacing: 0.16em
  • check headings, paragraphs and buttons for any text issues

The WCAG 2.2 skill behind this

1.4.12 Text Spacing - users must be able to adjust spacing without losing content or functionality.

Why this matters

Adjusted text spacing helps people with dyslexia, ADHD, cognitive load difficulties and low vision.

Try this bonus check

Apply text spacing on your mobile device. Does it still work correctly?
