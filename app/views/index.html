Elf-cessability induction at Santa HQ

Welcome to the annual festive review led by the Department of Elfish Affairs (DEA). In the run-up to Christmas, the DEA carries out a series of quick accessibility checks across the systems used by Santa HQ, from the Workshop Log to the Reindeer Health Monitor.

This year, you have been recruited as an Elf-cessability Specialist. You will work through a set of short, practical accessibility tasks that help make North Pole digital services more inclusive.

Each task should take around five minutes and focuses on a common accessibility issue found in real services. Some days you will spot issues by eye, and on other days you will use simple testing tools to help you.

The special elf testing tool

The elves use a special testing tool called the Web Accessibility Viewer for Elves (WAVE).

WAVE is a browser extension that highlights potential accessibility issues on a page, such as missing text alternatives, poor contrast, or structural problems.

You can install the WAVE browser extension from the WAVE website.

Automated tools like WAVE are helpful for finding issues quickly, but they do not catch everything. Throughout the 12 tasks, you will learn when tools are useful and when elf judgement is essential.
